jade-engine Introduction

jade-engine is a small, programmer-focused, 2D game engine written in C++ on top of SDL2.

Getting started

Prerequisites

  • Visual Studio - 15.7 or newer

Set-up

jade-engine is intended to be compiled directly into the target executable so there are no pre-built libraries to download and install.

Instead we recommend to set-up jade-engine as git submodule, include all the files in include and source folders and use provided Visual Studio Project Property files to pull in 3rd party dependencies. Detailed guide can be found in Installation page.

Usage

jade-engine comes with a samples solution that can be found in samples folder and should be the first stop when exploring jade-engine.

When browsing the documentation some of the key classes are: Game, IGameObject, IScene, Transform.
