void JadeEngine::LineBox::Show ( const bool  show)
overridevirtual

Show or hide game object.

@warn This does not affect any owned child game objects as this interface has no knowledge of them. It only handles tracking the visibility state. It is recommended to override this function for game object that own other game objects. Inside the overridden function one should call this - base - function and handle hiding/showing of children game objects there.

void SomeGameObject::Show(const bool shown) //override
{
IGameObject::Show(shown);
// Show or hide any child objects or perform any other necessary operations
_childObject->Show(IsShown());
}
